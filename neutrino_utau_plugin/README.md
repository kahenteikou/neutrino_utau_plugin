# NEUTRINO UTAU PLUGIN
## このプラグインについて
このプラグインはustをmusicxmlに変換するものです。
## 使い方
変換したいところを選択し、ツール → プラグイン -> Convert Music XML 
と選択しましょう。

するとファイル保存ダイアログが出てくるのでファイル名を入力して終了です.
